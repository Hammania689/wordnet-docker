

# wordnet-docker

WordNet Docker image

## **Usage:**

 1. **Clone repo:**
    
        git clone https://github.com/omarmohsen/wordnet-docker
    
  
 2. **Build Docker image:**
    
        sudo docker build -t wordnet wordnet-docker
    
 3.  **Run  WordNet:** 
    
          sudo docker run wordnet <args> 
      **e.g:**
      
          sudo docker run wordnet hub -synsn
